# ConvoBridge AI Multilingual App

ConvoBridge AI is a multilingual communication application that helps users communicate in different languages using AI technologies.

## Features
- Real-time translation
- AI chatbot
- Speech-to-text
- Multilingual support
- User-friendly interface

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB

## GitHub Actions
This project uses GitHub Actions for automated testing and workflow automation.
